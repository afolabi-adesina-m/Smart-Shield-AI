## Featured project

### [Smart-Shield AI](https://github.com/afolabi-adesina-m/Smart-Shield-AI)
Sheridan INFO53883 capstone — multimodal **highway safety scoring** for Ontario corridors.

- **NLP** alerts → text risk **T**
- **Vision** (ResNet18 ± autoencoder) → surface risk **V**
- **Environment** index → **E**
- Fusion → Safety Score **S** + speed advice

**Interactive showcase:** [afolabi-adesina-m.github.io/Smart-Shield-AI](https://afolabi-adesina-m.github.io/Smart-Shield-AI/)

```text
S = (w_T·T + w_V·V + w_E·E) × 100
```
