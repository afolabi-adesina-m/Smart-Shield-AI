# Trained models

Run `notebooks/capstone.ipynb` through the modeling sections to generate artifacts here.

| File | Description |
|------|-------------|
| `tfidf_vectorizer.joblib` | NLP Brain TF-IDF |
| `rf_tuned.joblib`, `lr_tuned.joblib` | Tabular classifiers |
| `scaler.joblib`, `feature_names.joblib` | Preprocessing |
| `dnn_smart_shield.pt` | PyTorch DNN |
| `vision_resnet18.pt` | Vision Brain ResNet18 |

Large `.pt` / `.joblib` files may be gitignored — retrain locally or use Git LFS for sharing.
